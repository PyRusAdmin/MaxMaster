# Max — программа для работы с OneMe API

## Требования

- **Python 3.x** (рекомендуется последняя стабильная версия)

## Установка

### 1. Установка Python

1. Скачайте установщик Python с официального сайта: https://www.python.org/downloads/
2. **Важно:** При установке обязательно поставьте галочку **"Add Python to PATH"** (Добавить Python в PATH)
3. Завершите установку

### 2. Установка зависимостей

Запустите файл **`Установка.bat`** — он автоматически установит все необходимые библиотеки:

- `websockets`
- `dotenv`
- `python-dotenv`

## Настройка

1. Создайте файл `.env` в папке с программой
2. Пропишите в нём номер телефона в Max и время задержки:
   ```
   PHONE_NUMBER=+7XXXXXXXXXXX
   SLEEP_BETWEEN_REQUESTS=2
   ```

**Параметры:**

- `PHONE_NUMBER` — номер телефона в международном формате (без +)
- `SLEEP_BETWEEN_REQUESTS` — задержка между запросами к API в секундах (рекомендуется 2-5 секунд)

## Получение номера для регистрации

SMS для регистрации можно получить на сервисе: **https://365sms.biz/activations**

## Запуск программы

Запустите файл **`Запуск.bat`** для старта программы.

## Структура проекта

- `main.py` — основной файл программы (поиск по телефону, экспорт в Excel)
- `read_file.py` — чтение номеров телефонов из файла
- `setup.py` — скрипт установки зависимостей
- `.env` — файл с настройками (номер телефона, задержка между запросами)
- `requirements.txt` — список зависимостей Python
- `cache/` — папка для хранения сессий аккаунта
- `input/` — папка для входных файлов (номера телефонов)
- `output/` — папка для результатов (Excel-файлы с данными пользователей)
- `logs/` — папка для логов программы
