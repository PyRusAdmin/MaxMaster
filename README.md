# Max — программа для работы с OneMe API

## Требования

- **Python 3.x** (рекомендуется последняя стабильная версия)

## Установка

### 1. Установка Python

1. Скачайте установщик Python с официального сайта: https://www.python.org/downloads/
2. **Важно:** При установке обязательно поставьте галочку **"Add Python to PATH"** (Добавить Python в PATH)
3. Завершите установку

### 2. Установка зависимостей

Запустите файл **`Установка.bat`** — он автоматически установит все необходимые библиотеки:
- `websockets`
- `dotenv`
- `python-dotenv`

## Настройка

1. Создайте файл `.env` в папке с программой
2. Пропишите в нём номер телефона в Max:
   ```
   PHONE_NUMBER=+7XXXXXXXXXXX
   ```

## Запуск программы

Запустите файл **`Запуск.bat`** для старта программы.

## Структура проекта

- `main.py` — основной файл программы
- `parse_chats.py` — парсинг чатов
- `saving_token.py` — сохранение токена сессии
- `setup.py` — скрипт установки зависимостей
- `cache/` — папка для хранения сессий аккаунта
